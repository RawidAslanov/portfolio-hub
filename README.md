# Portfolio Hub

Single-page portfolio with links to **live project demos** — for Fiverr gig and profile.

```powershell
cd "C:\Users\user\Desktop\portfolio-hub"
npm install
npm run dev
```

→ **http://localhost:3010**

## Deploy to Vercel

1. Create GitHub repo and push this folder
2. Import on [vercel.com](https://vercel.com)
3. Set environment variables (optional):

| Variable | Example |
|----------|---------|
| `NEXT_PUBLIC_SITE_URL` | `https://your-portfolio.vercel.app` |
| `NEXT_PUBLIC_FIVERR_URL` | `https://www.fiverr.com/your_username` |

4. Put the live URL in `FIVERR_GIG_EN.md` (replace `[PORTFOLIO_URL]`)
